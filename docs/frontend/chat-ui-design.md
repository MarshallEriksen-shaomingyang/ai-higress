# AI Higress Chat UI 设计规范：新中式数字水墨 (Digital Ink)

## 1. 核心视觉理念

Chat UI 不仅仅是信息的堆叠，更是一个**流动的对话空间**。我们采用“新中式数字水墨”风格，追求如宣纸般温润的质感与水墨晕染的呼吸感。

### 视觉特征：
- **墨色呼吸**：文字使用浓淡不一的灰阶（Ink System），避免纯黑。
- **琉璃质感**：对话框与侧边栏采用轻微的磨砂玻璃特效（Glassmorphism）。
- **留白之美**：对话气泡之间保持充足的间距，让视线呼吸。

## 2. 气泡设计 (Bubble Aesthetics)

### 2.1 用户气泡 (User Bubble)
- **背景**：Digital Blue 微渐变 (`#2563EB` -> `#1D4ED8`)。
- **圆角**：`rounded-2xl` (16px)，右下角适当收尖以指示来源。
- **阴影**：微弱的蓝色弥散阴影。

### 2.2 AI 气泡 (AI Bubble)
- **背景**：纯白 (`#FFFFFF`) 或极淡冷灰 (`#F9FAFB`)。
- **边框**：无边框，通过 `box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.02)` 浮起。
- **文字**：浓墨 (`#1F2937`)，正文使用淡墨 (`#4B5563`)。

## 3. 交互与动态 (Motion)

- **逐字入场**：AI 回复时采用优雅的平滑打字机效果。
- **气泡浮动**：新消息进入时，整体列表向上平滑错开淡入（Staggered Animation）。
- **状态指示**：AI 思考时，下方显示三个呈“呼吸感”淡入淡出的墨色圆点（Glow Dots）。

## 4. 侧边栏与历史记录

- **质感**：`backdrop-filter: blur(12px); background: rgba(255, 255, 255, 0.8);`
- **列表项**：鼠标悬停时，背景变为淡蓝色 (`#EFF6FF`)，左侧出现 3px 宽的蓝色书签线。

## 5. 实现建议 (Tech implementation)

- **Framer Motion**：用于处理气泡入场动画和列表重排。
- **Tailwind CSS**：利用 `glass` 变体和自定义阴影。
- **React Server Components**：消息历史列表服务端渲染，交互输入框客户端渲染。
