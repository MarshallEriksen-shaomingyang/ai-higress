## 设计细节说明

### 颜色方案

#### 404 页面
- **404 数字**: 渐变色 `bg-gradient-to-r from-primary to-primary/50`
- **标题**: `text-foreground` (默认文本色)
- **描述**: `text-muted-foreground` (次要文本色)
- **主按钮**: `bg-primary text-primary-foreground`
- **次按钮**: `border bg-background hover:bg-accent`
- **卡片**: `bg-card border shadow-sm`

#### 500 页面
- **警告图标**: `text-destructive` (红色/警告色)
- **标题**: `text-foreground`
- **描述**: `text-muted-foreground`
- **错误 ID 卡片**: `bg-card border`
- **代码块**: `bg-muted text-muted-foreground`
- **按钮**: 与 404 页面相同

### 字体大小

#### 404 页面
- **404 数字**: 
  - 桌面: `text-9xl` (128px)
  - 平板: `text-8xl` (96px)
  - 移动: `text-7xl` (72px)
- **标题**: `text-3xl` (30px)
- **描述**: `text-base` (16px)
- **快捷链接标题**: `text-base font-semibold`
- **快捷链接描述**: `text-sm` (14px)

#### 500 页面
- **警告图标**: 
  - 桌面: `size-32` (128px)
  - 平板: `size-24` (96px)
  - 移动: `size-20` (80px)
- **标题**: `text-4xl` (36px)
- **描述**: `text-base` (16px)
- **错误 ID**: `text-xs` (12px) monospace
- **支持文本**: `text-sm` (14px)

### 间距规范

#### 404 页面
- **整体容器**: `p-4` (16px padding)
- **内容区域**: `max-w-2xl` (672px max width)
- **元素间距**: `space-y-8` (32px vertical spacing)
- **按钮间距**: `gap-4` (16px gap)
- **快捷链接网格**: `gap-4` (16px gap)
- **卡片内边距**: `p-6` (24px padding)

#### 500 页面
- **整体容器**: `p-4` (16px padding)
- **内容区域**: `max-w-xl` (576px max width)
- **元素间距**: `space-y-8` (32px vertical spacing)
- **按钮间距**: `gap-4` (16px gap)
- **卡片内边距**: `p-6` (24px padding)

### 圆角规范
- **按钮**: `rounded-md` (6px)
- **卡片**: `rounded-xl` (12px)
- **代码块**: `rounded` (4px)

### 阴影规范
- **卡片默认**: `shadow-sm`
- **卡片悬停**: `shadow-lg`
- **按钮**: 无阴影（使用边框和背景色）

### 动画效果

#### 404 页面
1. **页面淡入**
   - 动画: `animate-in fade-in duration-500`
   - 效果: 整个页面从透明到不透明

2. **404 数字**（可选）
   - 动画: `animate-in zoom-in duration-700`
   - 效果: 从小到正常大小

3. **卡片悬停**
   - 过渡: `transition-shadow duration-300`
   - 效果: 阴影从 sm 到 lg

#### 500 页面
1. **页面淡入**
   - 动画: `animate-in fade-in duration-500`
   - 效果: 整个页面从透明到不透明

2. **警告图标脉动**
   - 动画: `animate-pulse`
   - 效果: 图标透明度周期性变化

3. **刷新按钮旋转**（点击时）
   - 动画: `animate-spin`
   - 效果: 图标旋转

### 交互状态

#### 按钮状态
- **默认**: 正常显示
- **悬停**: 背景色变化 + 轻微抬升
- **按下**: 轻微缩小
- **禁用**: 透明度 50% + 禁止点击

#### 卡片状态
- **默认**: `shadow-sm`
- **悬停**: `shadow-lg` + 轻微抬升
- **点击**: 导航到目标页面

#### 复制按钮状态
- **默认**: Ghost 样式
- **悬停**: 背景色变化
- **点击**: 显示 Toast 提示

## 响应式断点

### Tailwind 断点
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px
- `2xl`: 1536px

### 使用的断点
- **移动**: `< md` (< 768px)
- **平板**: `md` 到 `lg` (768px - 1023px)
- **桌面**: `≥ lg` (≥ 1024px)

## 可访问性标注

### ARIA 标签
```html
<!-- 404 页面 -->
<main role="main" aria-labelledby="error-title">
  <h1 id="error-title" aria-label="404 Page Not Found">404</h1>
  <nav aria-label="Quick navigation links">
    <!-- 快捷链接 -->
  </nav>
</main>

<!-- 500 页面 -->
<main role="main" aria-labelledby="error-title">
  <div role="alert" aria-live="assertive">
    <h1 id="error-title">Server Error</h1>
  </div>
  <section aria-label="Error details">
    <!-- 错误详情 -->
  </section>
</main>
```

### 键盘导航
- Tab 键: 按钮和链接之间导航
- Enter/Space: 激活按钮
- Escape: 关闭任何打开的对话框（如果有）

### 屏幕阅读器
- 所有图标都有 `aria-label` 或配套文本
- 错误 ID 可以被完整读取
- 按钮有清晰的标签

## 主题适配

### 亮色主题
- 背景: 白色/浅灰
- 文本: 深色
- 卡片: 白色带边框
- 按钮: 蓝色主色调

### 暗色主题
- 背景: 深色
- 文本: 浅色
- 卡片: 深灰色带边框
- 按钮: 蓝色主色调（调整亮度）

所有颜色使用 Tailwind 的语义化颜色类，自动适配主题。

## 图标使用

### Lucide React 图标
- `Home`: 首页图标
- `ArrowLeft`: 返回图标
- `RefreshCw`: 刷新图标
- `AlertTriangle`: 警告图标
- `Copy`: 复制图标
- `LayoutDashboard`: 仪表盘图标
- `Plug`: 提供商图标
- `BookOpen`: 文档图标
- `MessageCircle`: 支持图标

所有图标大小统一使用 `h-4 w-4` (16px) 或根据上下文调整。
